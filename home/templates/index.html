About


What is the purpose of this site?

The purpose of this site is to merge all the prices into one so people trying to buy gold have the possibility to compare prices and services. Also, this might increase the price competition between sellers, since now they can see their prices compared to others.

What does each column mean?

  • Price is the price per million of in-game currency. For example, if it says "0.59" it means that the seller is selling each in-game million coins for 0.59 USD.
  • Seller is the name of the seller in that website. Some websites have multiple sellers, some are owned by the seller itself.
  • Site
    • is both the name and a link to the website that has that price.

How are the prices fetched?

Every Hour I run a program (called a scraper) that loads the target website and finds the price that the seller is selling for.

Do I plan improving the website/adding more stuff?

The website is far from finished. Planning on adding statistics, "Selling" price (which not every site has it public), real time updating the price, a way for sellers to change the price here manually and possibly sort the sellers by trust.

Do I plan on monetizing?

I don't know yet. Planning on placing ads, selling the access to my database/API or making a Discord bot and sell that. If you want to make an offer I'm all ears. Just contact me on the contact page.

Is this website Open Source?

So far only the Frontend of this project is Open Source, and I don't plan on Open Sourcing the Scrapers or the API source until I decide how I'm going to monetize this website. If you want to check the progress or contribute to the Frontend part of this project you can check Here.
